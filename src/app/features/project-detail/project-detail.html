Descripción

Desarrollo integral de una plataforma web interactiva diseñada para la simulación de exámenes del ciclo formativo ASIR (Administración de Sistemas Informáticos en Red).

El sistema permite a los estudiantes evaluar sus conocimientos técnicos mediante test modulares con corrección en tiempo real, estadísticas de aciertos y explicaciones detalladas.

Arquitectura Técnica

  • > Frontend Core: HTML5 Semántico y JavaScript (Vanilla ES6+).
  • > UI/UX: CSS3 moderno con Flexbox/Grid, animaciones fluidas y diseño responsivo 'Glassmorphism'.
  • > Gestión de Estado: Implementación personalizada sin frameworks pesados, optimizando la carga del DOM.
  • > Carga de Datos: Sistema modular basado en ficheros JSON/JS para facilitar la escalabilidad de las preguntas.
